MRI Brain Tumor Image Segmentation

Project Overview

This project addresses the critical need for accurate and efficient brain tumor detection. The primary goal was to build a deep learning model capable of segmenting gliomas from multi-modal MRI scans. By leveraging the ResUNet architecture, which combines Residual Networks and U-Net, the model achieves high precision in identifying tumor boundaries. This automated approach aims to reduce the diagnostic workload on radiologists and improve the speed of treatment planning.

Key Features

  • Developed a deep learning model using the ResUNet architecture for semantic segmentation.
  • Implemented extensive image preprocessing and data augmentation to enhance model robustness.
  • Trained and validated the model on a publicly available dataset of brain MRI scans (BraTS).
  • Evaluated performance using critical metrics like the Dice Coefficient and Intersection over Union (IoU).

Technology Stack

  • TensorFlow & Keras
  • OpenCV
